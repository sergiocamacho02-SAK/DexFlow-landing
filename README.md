# DexFlow Sniper | Landing Page Profesional

Landing page desarrollada para el ecosistema Solana, con un enfoque en **Dark Luxury** y alto rendimiento web.

## 🚀 Tecnologías Implementadas
- **Frontend**: HTML5 Semántico, CSS3 (con variables globales y Glassmorphism).
- **Lógica**: JavaScript (ES6+) para validación dinámica y persistencia de datos.
- **Optimización**: Diseño responsive, SEO técnico y accesibilidad (ARIA).

## 🛠️ Características Técnicas (Proceso de Desarrollo)
- **Validación en Tiempo Real**: Implementación de feedback visual (glow/pulse) mediante validación dinámica.
- **Persistencia Local**: Uso de `localStorage` para mejorar la experiencia de usuario (UX).
- **Control de Versiones**: Desarrollo iterativo basado en 10 commits técnicos con hitos claros (Issues).
- **Seguridad**: Sanitización de inputs y mejores prácticas en el manejo de formularios.

## 📂 Estructura del Proyecto
- `index.html`: Estructura principal y SEO Meta-tags.
- `style.css`: Estilos modernos, variables globales y animaciones.
- `main.js`: Lógica de validación, persistencia de datos y estados de carga.
